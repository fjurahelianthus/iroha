# IROHA 設計總結

## 實作
- 語言：Python
- Parser：lark（LALR模式）
- 目標：interpreter

---

## 字符集分層

| 字符類型 | 角色 |
|----------|------|
| 漢字 | 保留類型名、builtin名、identifier的一部分 |
| 平假名 | 語法關鍵字和助詞，全部reserved |
| 片假名 | identifier的一部分（可與漢字混寫） |
| 全角符號 | 語法符號（（）「」。、） |

Identifier規則：漢字和片假名的任意組合，longest match。純漢字查保留字列表。函數identifier必須以する/す結尾。

## 類型系統

| 類型名 | 意義 |
|--------|------|
| `数` | 整數 |
| `浮` | 浮點數 |
| `符` | 字符 |
| `符々` | 字串 |
| `判` | Boolean（值為`是`/`非`） |
| `数々` | 一維整數陣列 |
| `数々々` | 二維整數陣列（々可無限疊加） |

## Binding

```
xは数の1です。       # 顯式類型binding
xは数です。          # 宣告不賦值
常に数の1。          # const
```

---

## 函數

```
# 宣告
計算するの法は数のa、数のbと数のcを使え。

# 定義
計算するとは、
  aとbをたす
  と答える。

# 呼叫
aとbを計算する。
a、bとcを計算する。
```

> ⚠️ `a、b、cを計算する。` → warning: もう噛んじゃう

---

## 控制流

```
# 範圍迴圈
1から10まで2ずつ繰り返す。

# while
xは是まで繰り返す。

# 條件分支
xは1でしたら、
  「一」と言う。
それとも xは2でしたら、
  「ニ」と言う。
それ以外、
  「いいえ」と言う。
```

## 內置動詞

| 動詞 | 意義 |
|------|------|
| `と言う` | print（複合token） |
| `と答える` | return（複合token） |
| `をたす` | 加法 |
| `をへる` | 減法 |
| `をかける` | 乗法 |
| `をわる` | 除法 |
| `をみる` | debug inspect |
| `をきく` | 讀取輸入 |

---

## 運算子

**比較：**
- `xはyより` → x < y
- `xはyより等` → x ≤ y
- x > y → 交換xy位置

**邏輯/Bitwise：**
| 符號 | 意義 |
|------|------|
| `且` | AND |
| `又` | OR |
| `の逆` | NOT（後置） |
| `異` | XOR |
| `動` | shift |

**Grouping：** `（x且y）の逆`

---

## Lexer 特殊處理

- `と言う`、`と答える` → 整體tokenize為複合token，消除と的歧義
- 平假名序列 → 直接tokenize為關鍵字
- 漢字片假名混寫 → longest match為IDENTIFIER
- する/す結尾 → FUNC_IDENTIFIER

## 報錯風格

```
エラー：xは知らない。       # 未定義變數
エラー：これは違います。     # 類型錯誤
エラー：だめです。           # 非法操作
警告：もう噛んじゃう         # 全頓號分隔參數
```
